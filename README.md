# No Zombies Mod for Cataclysm: Dark Days Ahead

A mod that removes all vanilla zombies, including fungus zombies, from the game.
The mod is made by lcarus and has been updated by me, Countern, to remove all the new zombies added since the last mod update.

### Download & Installation Instructions

1. Remove the old or not updated No Zombies Mod
2. Click the green `Code` button then the `Download ZIP` button or [click here](https://www.github.com/Countern/nozombies/archive/master.zip) to download the mod.
3. Move the `nozombies-master` folder to your Cataclysm DDA mods folder
4. Enable the mod when creating a new world
